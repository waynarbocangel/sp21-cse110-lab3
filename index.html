What happened today!

Today was an intense day! I thought it would never end. I had to finish the implementation of the toolbar in my app Alexandria, but I realized that I had to design the bar progamatically instead of through an Xcode storyboard. Doing so has taken me 6 hours so far and today is the second day I'll have to go to sleep because I was not able to finish the task on time. Developing software is challenging at every step, specially when someone is doing it all by himself. Still, knowing that whatever you write in your text editor can eventually become usefull to someone else is extremely inspiring. Software Engineering is like a day that is rainy and sunny at the same time, which is funny since that is what happend to day in my hometown La Paz, Bolivia.
